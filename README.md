# Data Cleaning & Reporting Automation

**Thiranex Internship - Project 4**

## Project Overview
This project automates the data cleaning and reporting process for sales data using Python and Pandas.

## What I Did
1. Cleaned messy sales data - removed duplicates, filled missing values
2. Standardized inconsistent region names
3. Created automation script that runs in 1 click
4. Generated region-wise sales report and bar chart

## Files Included
- `Data_Cleaning_Reporting_Automation_Project4.ipynb` - Main Python Code
- `Cleaned_Data_2026-07-04.csv` - Cleaned Dataset
- `Cleaned_Sales_Report_2026-07-04.csv` - Final Report
- `Sales_Report_Chart.png` - Sales Visualization

North region had the highest total sales. Automation reduced manual work by 90%.
